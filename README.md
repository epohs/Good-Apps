# Running list of apps that are good for


## General

- [Noir](https://getnoir.app)
  - Dark mode for Safari.


## MacOS

- [Homebrew](https://brew.sh)
  - Commandline package manager.

- [Transmit](https://panic.com/transmit/)
  - Best FTP app for Mac.

- [Ghostty](https://ghostty.org)
  - Easy, fast, native Terminal.app replacement.

- [Tot](https://tot.rocks)
  - Small, clean scratchpad/notes applet.

- [Amphetamine](https://apps.apple.com/app/amphetamine/id937984704?mt=12)
  - Keep your Mac awake.


## iPad

- [Textastic](https://www.textasticapp.com)
  - Text editor with built-in FTP and SSH.
  
- [Terminus](https://termius.com)
  - Excellent SSH terminal.
  
- [Working Copy](https://workingcopy.app)
  - Git client that integrates with native Files app.


## Linux

- [Micro](https://micro-editor.github.io)
  - Commandline text editor.
  
- [btop](https://github.com/aristocratos/btop)
  - Nicer, prettier top.
