# Running list of apps that are good.

This is an ongoing list that will change over time. Most of these are free apps, or have free versions. Some of them are paid. Theyâ€™re all good.


## General

- [Noir](https://getnoir.app)

  - Dark mode for Safari.

- [1Blocker](https://1blocker.com)
  
  - Ad blocking for Safari.


## MacOS

- [Homebrew](https://brew.sh)

  - Command line package manager.

- [Transmit](https://panic.com/transmit/)

  - Best FTP app for Mac.

- [Ghostty](https://ghostty.org)

  - Easy, fast, native Terminal.app replacement.

- [Tot](https://tot.rocks)

  - Small, clean scratchpad/notes applet.

- [Amphetamine](https://apps.apple.com/app/amphetamine/id937984704?mt=12)

  - Keep your Mac awake.


## iPad/iPhone

- [Textastic](https://www.textasticapp.com)

  - Text editor with built-in FTP and SSH.
  
- [Terminus](https://termius.com)

  - Excellent SSH terminal.
  
- [Working Copy](https://workingcopy.app)

  - Git client that integrates with native Files app.


## Linux

- [Midnight Commander](https://midnight-commander.org)
  
  - Visual terminal file manager. 
 
- [btop](https://github.com/aristocratos/btop)

  - Nicer, prettier top.

- [Micro](https://micro-editor.github.io)

  - Terminal text editor.
  
- [uv](https://github.com/astral-sh/uv)

  - Python package manager (replacement for pip and venv).

- [Fastfetch](https://github.com/fastfetch-cli/fastfetch)

  - Display a pretty, and useful welcome screen for terminal sessions.
  
- [LiteCLI](https://litecli.com)

  - Command line SQLite interface.
